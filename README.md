This repository contains a Machine Learning project focused on predicting the likelihood of heart disease in patients based on their medical attributes. The goal is to build a reliable classification model that can assist healthcare professionals in early detection.

---

## 📌 Project Overview
Heart disease is one of the leading causes of mortality globally. Early diagnosis plays a crucial role in preventing severe complications. This project utilizes data science techniques to preprocess clinical data, perform Exploratory Data Analysis (EDA), and train Machine Learning models to classify whether a patient has heart disease or not.

## 📊 Dataset Attributes
The dataset typically includes the following clinical features:
* **Age:** Age of the patient
* **Sex:** Gender (1 = male; 0 = female)
* **Chest Pain Type (cp):** Type of chest pain experienced (Value 0 to 3)
* **Resting Blood Pressure (trestbps):** Resting blood pressure in mm Hg
* **Cholesterol (chol):** Serum cholesterol in mg/dl
* **Fasting Blood Sugar (fbs):** (1 = true; 0 = false) if > 120 mg/dl
* **Resting ECG (restecg):** Resting electrocardiographic results
* **Max Heart Rate (thalach):** Maximum heart rate achieved
* **Exercise Induced Angina (exang):** (1 = yes; 0 = no)
* **ST depression (oldpeak):** ST depression induced by exercise relative to rest
* **Target:** Diagnosis of heart disease (1 = Defect, 0 = Normal)

## 🚀 Key Features & Workflow
1. **Data Cleaning:** Handling missing values, removing duplicates, and data transformation.
2. **Exploratory Data Analysis (EDA):** Visualizing feature distributions, correlation matrices, and identifying patterns.
3. **Feature Engineering:** Scaling numerical features and encoding categorical variables.
4. **Model Training:** Implementing classification algorithms (e.g., Logistic Regression, Random Forest, or Support Vector Machines).
5. **Evaluation:** Assessing models using Accuracy, Precision, Recall, F1-score, and Confusion Matrix.
